

# 05 - Why Do Traders Use Different EMAs?

## Goal

Understand why traders use different EMA periods and what each one tells us.

---

# What Does EMA Number Mean?

The number means how many candles the EMA looks at.

Example:

On a 1-hour chart:

* EMA20 = last 20 hourly candles
* EMA50 = last 50 hourly candles
* EMA200 = last 200 hourly candles

Bigger number = slower reaction.

Smaller number = faster reaction.

---

# Different EMA Speeds

## EMA9 + EMA21 (Very Short-Term)

Used by:

* Scalpers
* Day traders

Purpose:

Find fast momentum changes.

Example:

```text
EMA9
   /
  /

EMA21
---------
```

EMA9 above EMA21:

Short-term buyers are getting stronger.

---

## EMA20 + EMA50 (Short / Medium-Term)

Used by:

* Swing traders
* Trend traders

Purpose:

Find the current trend.

Example:

```text
EMA20
    /
   /

EMA50
---------
```

EMA20 above EMA50:

Short-term trend is bullish.

---

## EMA50 + EMA200 (Medium / Long-Term)

Used by:

* Long-term traders
* Investors

Purpose:

Find the bigger market direction.

Example:

```text
EMA50
     /

EMA200
----------
```

EMA50 above EMA200:

The long-term trend is usually considered bullish.

---

## EMA20 + EMA200 (Short vs Long-Term)

Purpose:

Compare current movement with the big picture.

Example:

```text
EMA20
   ↑

EMA200
---------
```

Meaning:

Short-term buyers are strong, but we check if the long-term trend agrees.

---

# EMA Alignment (Strong Trend)

When multiple EMAs agree:

Example:

```text
Price
  ▲

EMA9
  ▲

EMA21
  ▲

EMA50
  ▲

EMA200
```

This means:

* Very short-term trend ✅
* Short-term trend ✅
* Medium-term trend ✅
* Long-term trend ✅

Many traders call this a strong trend environment.

---

# Important Example

Sometimes:

```text
EMA9 ↑
EMA21 ↑
EMA50 ↑
EMA200 ↓
```

Meaning:

Short-term:

Bullish

Long-term:

Still bearish

The market may be recovering, but the big trend has not changed yet.

---

# Simple Analogy 🚗

* EMA9 = looking through the front window (very fast changes)
* EMA21 = nearby road
* EMA50 = highway direction
* EMA200 = entire journey direction

The smaller EMAs turn first.

The bigger EMAs turn later.

---

# Remember

✅ EMA9/21 → Fast momentum

✅ EMA20/50 → Trading trend

✅ EMA50/200 → Long-term trend

✅ EMA200 = big picture

✅ EMA does not predict the future

✅ EMA gives market information
