 # Aegis Trading System (ATS)

> A professional rule-based algorithmic trading platform built with Python, FastAPI, React, and PostgreSQL.

---

# Overview

Aegis Trading System (ATS) is a modular trading platform designed to analyze multiple financial markets including:

- Indices
- Forex
- Commodities
- Cryptocurrencies
- Stocks

ATS uses a rule-based multi-timeframe decision engine to detect market conditions, identify high-probability trade opportunities, manage risk, and execute trades.

The platform is designed for long-term scalability, maintainability, and future expansion.

---

# Core Philosophy

Accuracy > Speed > Profit

Protecting trading capital always comes before chasing profits.

Every decision made by ATS should be explainable, testable, and repeatable.
